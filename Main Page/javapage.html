Java

What is Java?

Java is a type of object-oriented programming language, widely used for developing websites and applications. Its intention is to allow developers to write the program only once and run it through any device.

JSP is the acronym for Java Server Pages, it is an open specification scripting language with the primary objective of generating dynamic content for Internet pages.

How does it work?

In web pages JSP (Java Server Pages) is used, instead of using HTML to develop static and featureless web pages, JSP is implemented to create dynamism. It is possible to write HTML, with embedded JSP code. As HTML is a static language, JSP will be responsible for creating dynamism. The JSP needs server to work for being a language Server-side script, the user can not see the JSP coding, because this is converted directly by the server, being presented to the user only HTML coding.

A JSP page has jps extension and consists of a page with HTML coding and Java coding, inserted between the tag's <% and %>, called scriptlets and working as follows: the server receives a request for a JSP page, interprets this page generating the HTML coding and returns to the client the result of your request
